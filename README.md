# MonoPlat

MonoPlat is a simple platformer game with custom pixel art, the visuals are tailored to be dull and boring, so that when you beat the final level the player is escaping the dull, boring world it's stuck in.

The game features around 10 levels with double jumping, dashing, 0-gravity, hazards, and mazes, reach the flag to advance to the next level. The controls and object explanation can be found from the buttons on the homepage, which will redirect you to explanation pages.
